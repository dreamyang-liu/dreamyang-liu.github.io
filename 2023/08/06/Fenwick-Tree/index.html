Fenwick Tree

Posted on 2023-08-06 Edited on 2023-08-08 In Algorithm

Fenwick Tree, also known as binary indexed tree, is a data structure that supports point updates and range queries. It is based on the assumption that any prefix \([1,n]\) can be decomposed into at most logn ranges, such that the information of these logn ranges is known.

As we can see from the above figure, the right boundary of the range controlled by each position \(c_i\) is \(i\). In Fenwick, \(c_i\) is assigned to control a range of length \(2^k\). Here, \(k\) is the number of the first 1 bit from the right, minus 1. For example, \(12 = (1100)_2\), then \(k = 2\), and \(2^k = 4\). So in this case, the range controlled by \(c_{12}\) is \([12 - 4 + 1, 12]\). \(2^k\) can be obtained using the lowbit, which is \(x\ \& \ (-x)\) .
